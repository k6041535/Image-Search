# NTU-Multimedia
static - images(存放結果圖), imgupload(上傳的圖片)
templates - 存放html檔（index.html、show.html、upload.html）
app.py - 主程式
test.py - 測試程式
環境 - Python 3.8.5, Flask 1.1.1, Opencv 4.0.1
使用方法 - 先上傳圖片再選擇images檔案夾就可看到結果
